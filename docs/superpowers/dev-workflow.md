# 开发流程

---

### brainstorming

> 创意构思与设计探索

- **来源**：[obra/superpowers](https://github.com/obra/superpowers)
- **本地路径**：`docs/skills/superpowers/brainstorming/`
- **适用场景**：在任何创建性工作（创建功能、构建组件、修改行为）之前，探索用户意图、需求和设计
- **重要**：这是 Superpowers 中最重要的 Skill，**必须**在任何创建性工作前使用

**使用示例：**
```
"我想做一个用户仪表盘，帮我先梳理一下需求和设计方案"
```
