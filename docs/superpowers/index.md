# Superpowers 插件 Skills

Superpowers 是一个 Codex 插件，提供软件开发全流程的 Skills，覆盖从构思到交付的每个环节。

来源：[obra/superpowers](https://github.com/obra/superpowers)

## 分类导航

| 分类 | 数量 | 说明 |
| --- | --- | --- |
| [开发流程](/superpowers/dev-workflow) | 8 | 头脑风暴、计划、TDD、调试、并行任务等 |
| [代码评审与 Git](/superpowers/code-review-git) | 4 | 代码评审、分支管理、Git Worktree |
| [Skill 管理](/superpowers/skill-management) | 2 | 编写 Skill、使用 Superpowers |

## 安装方式

从 GitHub 安装 Superpowers 插件：

```bash
git clone https://github.com/obra/superpowers.git ~/.codex/plugins/superpowers
```

或使用离线文件：`docs/skills/superpowers/`

## 推荐使用顺序

Superpowers 的 Skills 设计为协同工作，推荐流程：

1. `brainstorming` — 先构思和探索想法
2. `writing-plans` — 写实现计划
3. `test-driven-development` — TDD 方式开发
4. `systematic-debugging` — 遇到问题时系统化调试
5. `verification-before-completion` — 完成前验证
6. `requesting-code-review` — 请求代码评审
7. `finishing-a-development-branch` — 完成分支合并
