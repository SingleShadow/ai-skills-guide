# 浏览器与桌面控制

通过 Codex 控制浏览器和桌面应用的 Skills。

---

### control-in-app-browser

> Codex 内置浏览器控制

- **来源**：[openai-bundled 插件](https://github.com/openai/codex)
- **本地路径**：`docs/skills/plugins/openai-bundled-control-in-app-browser/`
- **适用场景**：控制 Codex 应用内置的浏览器，进行网页浏览、截图、自动化操作

**使用示例：**
```
"在内置浏览器中打开 https://github.com 并截图"
```

### control-chrome

> Chrome 浏览器控制

- **来源**：[openai-bundled 插件](https://github.com/openai/codex)
- **本地路径**：`docs/skills/plugins/openai-bundled-control-chrome/`
- **适用场景**：控制用户的 Chrome 浏览器，利用已有登录状态和扩展

**使用示例：**
```
"在 Chrome 中打开新标签页，访问 Gmail 收件箱"
"用 Chrome 搜索 GitHub 上最近的 codex 相关仓库"
```
