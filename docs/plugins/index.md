# 插件生态 Skills

由 OpenAI 插件生态提供的 Skills，涵盖浏览器控制、设计视频、文档处理等领域。

## 分类导航

| 分类 | 说明 |
| --- | --- |
| [浏览器与桌面控制](/plugins/browser-control) | 控制 Codex 内置浏览器、Chrome、Windows 桌面应用 |
| [设计与视频](/plugins/design-video) | Canva 设计、HyperFrames 视频合成、GSAP 动画、Remotion |
| [文档处理](/plugins/document-processing) | Word、PDF、PowerPoint、Excel 文档处理 |

## 安装方式

插件 Skills 通常通过对应的插件安装。在 Codex 中说：

```
"使用 skill-installer 安装 [skill 名称]"
```

或使用离线文件：`docs/skills/plugins/`
