# 文档处理

办公文档（Word、PDF、PowerPoint、Excel）处理 Skills。

---

### documents

> Word 文档创建与编辑

- **来源**：[openai-primary-runtime 插件](https://github.com/openai/codex)
- **本地路径**：`docs/skills/plugins/openai-primary-runtime-documents/`
- **适用场景**：创建、编辑 .docx 文件，支持修订、批注、格式保留

**使用示例：**
```
"创建一份 Word 合同文档，包含甲乙双方信息、条款和签名栏"
"编辑这份 Word 文档，将第三段的日期改为 2026 年 7 月"
```
