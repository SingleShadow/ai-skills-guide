# 系统内置 Skills

Codex 运行时自带的 Skills，无需额外安装即可使用。

---

### imagegen

> AI 图像生成与编辑

- **来源**：Codex 系统内置
- **本地路径**：`docs/skills/system/imagegen/`
- **适用场景**：生成照片、插画、纹理、精灵图、透明背景图片等

**使用示例：**
```
"生成一张日落时分的海滩风景照片"
"把这张图片的背景换成纯白色"
```
