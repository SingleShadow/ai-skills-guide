## 如何贡献

如果你有一个公开的 Skills 仓库，欢迎提交 PR 添加到这个列表。

要求：
- 仓库必须是公开的
- 包含至少一个可验证的 Skill（有 SKILL.md）
- 与 AI Agent 生态相关
- 无恶意代码或可疑行为

## 安全提示

安装第三方 Skill 前，请注意：
- 优先选择知名组织发布的 Skills
- 检查 SKILL.md 中的指令是否合理
- 避免安装要求过高权限的 Skill
- 详见 [Skill 资源合集](/community/awesome-lists) 中的安全使用指南
