# last30days Skill - AI 舆情调研

> 130k+ Stars | 深度调研过去 30 天内人们对任何话题的真实讨论

[mvanhorn/last30days-skill](https://github.com/mvanhorn/last30days-skill) 是一个强大的 AI 舆情调研 Skill，能够从 Reddit、X/Twitter、YouTube、TikTok、Hacker News、Polymarket、GitHub 等多个平台抓取过去 30 天内关于任意话题的真实讨论内容和互动数据。

## 核心能力

| 能力 | 说明 |
| --- | --- |
| **多平台覆盖** | Reddit、X/Twitter、YouTube、TikTok、Hacker News、Polymarket、GitHub、Bluesky |
| **趋势发现** | 发现正在兴起的讨论、产品反馈、用户痛点 |
| **情绪分析** | 理解用户对特定话题的真实情绪和观点 |
| **竞品监控** | 追踪竞品在过去 30 天内的社区讨论 |
| **信号捕捉** | 捕捉招聘信号、技术趋势、市场变化 |

## 支持的数据源

- Reddit (帖子、评论、投票)
- X / Twitter (推文、转发、互动)
- YouTube (视频、评论、观看量)
- TikTok (视频、标签趋势)
- Hacker News (帖子、评论、投票)
- Polymarket (预测市场数据)
- GitHub (Issues、PR、讨论)
- Bluesky (帖子、互动)

## 使用示例

```bash
# 调研 AI 视频工具的最新动态
last30days AI video tools

# 了解 React 社区最近在讨论什么
last30days what users want in react

# 追踪 Nvidia 财报后的市场反应
last30days nvidia earnings reaction
```
